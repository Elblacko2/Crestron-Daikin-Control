# Crestron-Daikin-Control
Crestron Simpl+ Module that uses Web request to control Daikin airco units on your network

![Image of Module](https://github.com/Elblacko2/Crestron-Daikin-Control/blob/main/Module.png)

### Current features:
* Power control
* Opparation mode
* Temperature Control
